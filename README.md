# FedRAMP Resources

This repository is provided by the [OSCAL Foundation](https://oscalfoundation.org) in support of OSCAL-based FedRAMP artifacts.

It provides the following in OSCAL XML, JSON, and YAML formats:
- FedRAMP baselines
- Example System Security Plan (SSP)
- Example Plan of Action and Milestones (POA&M) [pending]
- Example Security Assessment Plan (SAP) [pending]
- Example Security Assessment Report (SAR) [pending]

Additional resources will be added as they become available.

All FedRAMP community members are welcome to provide sample OSCAL content for their specific FedRAMP use cases.
